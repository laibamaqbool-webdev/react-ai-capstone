# AI-Assisted Development Setup Project

## Overview

This repository is created for the AI-assisted development track setup phase. The purpose of this project is to prepare a complete development environment and learn how to use AI tools effectively during the software development workflow.

## Development Environment

The project setup includes:

* Node.js (LTS)
* Git
* GitHub
* VS Code
* Claude Code

## Project Setup

The setup phase includes:

* Installing and configuring development tools.
* Creating and managing a GitHub repository.
* Adding project documentation files.
* Setting up Git version control.
* Following Conventional Commits standards.

## AI-Assisted Development

Claude Code is used as an AI assistant to support development tasks, review documentation, suggest improvements, and improve the overall development workflow.

## Repository Files

* `README.md` - Project documentation and overview.
* `LICENSE` - Project license information.
* `.gitignore` - Files and folders excluded from Git tracking.
* `CLAUDE.md` - Instructions and guidelines for AI-assisted development.

## Git Commit Convention

This project follows Conventional Commits format.

Examples:

* `chore: initialize repository setup`
* `docs: add project documentation`
* `docs: add claude instructions`

## Goals

* Learn AI-assisted software development practices.
* Build a proper development workflow.
* Maintain clean documentation and Git history.
* Improve productivity using AI coding assistants.
